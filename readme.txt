Learn the Periodic Table one element at a time. This plugin will randomly display a Periodic Table element each day.

== Description ==

This plugin will randomly display one element from the Periodic Table each day. 

All 118 Periodic table elements are included. 

The plugin will display the element name, atomic number and symbol. 

Learn each element the easy way. After a few months you will be able to remember each element and its corresponding Atomic number and Symbol.

== Installation ==

Follow these steps to install the plugin.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the shortcode [periodic-table] to display the plugin on your website.
